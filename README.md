# library-inventory

## Operations v1
### Book API
* `POST /api/v1/book`
* `DELETE /api/v1/book`
* `GET /api/v1/book?title={}`
* `GET /api/v1/book/{id}`
  
### User API
* `POST /api/v1/user`
* `GET /api/v1/user?name={}`
* `GET /api/v1/user/{id}`

### Loan API
* `POST /api/v1/loan`
* `PUT /api/v1/loan`
